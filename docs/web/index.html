MydIAgram

Agente conversacional que genera diagramas de software editables a partir de lenguaje natural: entidad-relación, secuencia, diagramas de flujo, arquitectura (C4), mapas mentales y casos de uso. Esta es la documentación técnica del sistema: arquitectura, motor de generación, modelo de datos y funcionalidades.

React 19 + React Flow 12 Express 5 + Socket.io LangGraph + FastAPI Supabase · PostgreSQL 16
◆

Mapa del sistema

Visión de pájaro de los 3 procesos y su comunicación

El navegador solo habla con el backend por WebSocket. El backend es el único punto de acceso a datos: verifica el JWT, aplica límite de tasa y caché, y reenvía la petición al agente por HTTP recibiendo el diagrama en streaming NDJSON. El microservicio de IA nunca se expone al navegador.

Grafo de refinamiento
Loop ReAct: el agente edita el diagrama con herramientas (añadir, quitar y actualizar nodos y aristas). Escape hatch regenerate e interrupt/resume para aclaraciones.
